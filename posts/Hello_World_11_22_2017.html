This week, I've been looking through GitHub for projects that I can contribute to. I am most comfortable in Javascript, Java and Python, although R and Rudy (on Rails) are also really interesting. A project that I'm been perusing most recently is with Anadana Kapur, an amazing Indian film maker crafting an "interactive" film from footage she's captured of Indian women of various social backgrounds. This project is going to be technically challenging (she has terabytes of video that need to be displayed via both a desktop website as well as mobile platform!) as well as a significant creative undertaking. Anadana is passionate about creating an experience where the audience isn't merely just a passive observer, but instead can follow branching story lines that follow different character narratives. This project has just begun, but you can follow (and contribute to!) our progress on our GitHub: here.
